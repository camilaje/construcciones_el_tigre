Cuándo usar este formulario

  • Úsalo cuando llegue stock nuevo desde un proveedor externo — no cuando se traslada entre obras. Para traslados usa Registrar movimiento.
  • Si {{ routeData.itemLabel }} no aparece en la lista, agrégalo primero desde Catálogos antes de registrar la compra.
  • Obra / Bodega de destino es dónde quedará almacenado el stock que compraste. Si ya había unidades ahí, la cantidad se suma al stock existente.
  • Referencia / proveedor es opcional pero útil para rastrear facturas y conciliar compras con contabilidad.
  • La cantidad total del catálogo se actualiza automáticamente y el registro queda visible en el historial de movimientos como compra.
